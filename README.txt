CAMILA IN WONDERLAND — GITHUB + NETLIFY PROJECT

FILES
- index.html: Main invitation website
- styles.css: Design and responsive layout
- script.js: Storybook opening, countdown, menu, calendar download
- thank-you.html: Confirmation after an RSVP
- netlify.toml: Netlify publishing settings
- assets/trailer-poster.svg: Temporary trailer cover
- assets/trailer.mp4: Add your final trailer with this exact filename

HOW TO ADD THE TRAILER
1. Download the final Luma trailer to the iPad Files app.
2. Rename it to: trailer.mp4
3. Upload it into the assets folder in GitHub.
4. Netlify will automatically publish the updated site.

RSVP RESPONSES
After Netlify publishes the site, responses appear in:
Netlify Dashboard > Forms > camila-rsvp
